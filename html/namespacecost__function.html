Computes the marginal likelihood of the given hyperparameters and input output data.

This means the probability of the given parameters for the input-output data is determined. Maximizing this likelihood leads to the most likely parameters.

Parameters
xnumpy_matrix: input data
ynumpy_matrix: output data
lengthint: length of data set
sig_ffloat: parameter defining the process noise
lfloat: parameter defining the length scale
sig_nfloat: parameter defining the output noise
Returns
: probability of the given parameter for the input-output data
